# Quiz App (Kotlin + JavaFX)

A lightweight desktop quiz application built with **Kotlin** and **JavaFX**.  
It presents multiple-choice questions, tracks score in real time, and shows a final result summary.

## Features
- 10-question multiple-choice quiz
- Randomized question order on each run
- Score tracking with percentage result
- Simple, clean JavaFX UI
- Restart quiz option

## Tech Stack
- Kotlin (JVM)
- JavaFX
- Gradle
